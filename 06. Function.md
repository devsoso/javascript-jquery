### 함수
#### 1. 기본적인 함수의 정의와 호출

```javascript
function greet(){
  alert("안녕히가세요");
}
<button onclick="greet();">버튼</button>
```

#### 2. 매개변수가 있는 함수

```
function 함수명(매개변수1, 매개변수2){
  실행문;
}
함수명(인자값1, 인자값2);
```
```javascript
function greet(theText){
  alert(theText);
}
<button onclick="greet("반갑습니다");">버튼1</button>
<button onclick="greet("안녕히가세요");">버튼2</button>
```
